TXPipe Installation
==========

Requirements
------------

TXPipe requires python 3.6 or above.

The pipeline software ceci that TXPipe uses can be installed with:
``
    pip install ceci
``

Various stages of TXPipe also require:

``
    numpy
    scipy
    fitsio
    healpy
    astropy
``

which can all also be installed with pip.
